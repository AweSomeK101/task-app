
# Task App


Bearer Token required for all endpoints except user register and user login

#### ENV variables: 

 - MONGODB_URL
 - PORT
 - SALT - (random secret word for json web token)

  

## End-point: Register User

Method: POST

>```

> /api/user/register

>```

### Body (**raw**)

  

```json

{

"name":  "Angela kinsey",

"email":  "angela@samad.com",

"password":  "passwordis",

"phone":  7894561230

}

```
